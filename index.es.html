ideckia

ideckia es un sistema para controlar el ordenador mediante un smartphone o table como panel de control.

El panel es totalmente configurable, se pueden crear infinidad de directorios y se pueden poner tantos elementos como se quiera dentro de estos directorios: Se pueden crear automatizaciones para gaming, streaming, desarrollo de software, el hogar...

Multi plataforma

Funciona en GNU/Linux, Windows y OSX. Totalmente portable (<50MB). Descárgalo para tu SO:

Un montón de posibilidades

Ejecuta comandos, escribe contraseñas, crea atajos del teclado, silencia micro, controla OBS Studio, zonas horarias, tiempo, cuenta atrás, temporizador, registra tus horas de trabajo, crea colores aleatorios...

Lista de acciones
Sin límite de directorios

No hay número máximo de directorios. Crea paneles complejos con cuantos directorios quieras con los elementos que quieras para tenerlo todo bien ordenado.

Múltiples estados por item, múltiples acciones por estado

Cambia el estado (texto, icono...) de un item al hacer click. Ejecuta varias acciones por cada estado del item.

Elementos fijos

Ten siempre a mano los items necesarios, aunque explores distintos directorios.

Editor integrado

Si tienes un navegador web, el editor está ahí mismo.

Abrir editor
Diálogos

Puedes usar diálogos desde las acciones para interactuar con el usuario: textos, passwords, preguntas, calendarios...

Software libre

Software totalmente libre: la aplicación, el editor, el API, las acciones y la aplicación de smartphone está bajo la licencia GPL 2.
